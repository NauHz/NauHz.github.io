虚拟机搭建Hexo博客

下载VMware Workstation、镜像操作系统、Nodejs、Git

效验node：node -v、效验npm：npm -v

安装淘宝的cnpm 管理器：👇

效验cnpm：cnpm -v

安装hexo框架：👇

成功会在目录中看到.npminstall_tarball文件

查看hexo版本：hexo -v

创建blog目录：找个位置新建放blog的文件夹

初始化博客：在blog根目录右键点Git Bash Here输入hexo init

生成网站静态文件：hexo s

本地访问地址：(http://localhost:4000/)

下载butterfly主题到本地：👇

修改根目录的 _config.yml，把主题改为butterfly👇

根据butterfly博客根据自己需要装修博客：butterfly

在blog目录下安装git部署插件：cnpm install --save hexo-deployer-git

Github创建一个新的仓库： 你的Github名字.github.io

Github用户名：git config --global user.name "xxx"

Github邮箱：git config --global user.email "xxx"

配置_config.yml:👇

清除缓存：hexo clean

生成：hexo g

部署到Github仓库里：hexo d

访问地址即可查看博客：(https://你的Github名字.github.io/)

克隆仓库：git clone 仓库地址、code 后在根目录安装依赖 npm install、在主题目录重新安装一次hexo-theme-butterfly
